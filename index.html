Installation

Get up and running with FHE/CROSS in minutes.

bash 
pip install fhe-cross
# Or clone the repository
git clone https://github.com/example/fhe-cross.git

Quick Start

Initialize your first secure context.

python 
import fhe_cross as fc

# Initialize context
ctx = fc.Context(security_level=128)
print("Context initialized")
